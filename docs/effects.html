Effect Index

Every "drug" (pack) in PiHK.AI is a recipe of low-level effects: precise, reversible interventions installed into a running model at inference time — no fine-tuning, no weight edits. An effect is a forward hook, a LogitsProcessor, a KV-cache subclass, an MoE-router patch, or an image-generation transform.
