What is PiHK.AI?

Inspired by Alexander Shulgin's PiHKAL (Phenethylamines I Have Known and Loved), PiHK.AI represents a new frontier in AI research: the systematic exploration of neuromodulatory effects on large language models.

Quick Start

pip install neuromod-llm # Load a model and apply a pack from neuromod import PackRegistry, PackManager from transformers import AutoModelForCausalLM model = AutoModelForCausalLM.from_pretrained("microsoft/DialoGPT-small") registry = PackRegistry() pack = registry.get_pack("caffeine") # Try "alcohol", "mdma", "placebo" pack_manager = PackManager() pack_manager.apply_pack(model, pack) # Now your AI is "caffeinated"! ‚òï
